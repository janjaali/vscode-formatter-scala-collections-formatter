# scala-collections-formatter

Simple, opinionated and not-well tested formatter for scala collections.

## Development

Press `F5` to open a new window with the scala-collections formatter extension loaded.

## Package

[vsce](https://github.com/microsoft/vsce) is ued to package the extensions:

```shell
vsce package
```

You can find a detailed description about packaging, publishing and managing of VS Code extensions [here](https://code.visualstudio.com/api/working-with-extensions/publishing-extension).
